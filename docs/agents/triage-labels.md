# Triage labels

Engineering skills 使用五个 canonical triage role。本文件将这些 role 映射到本仓库 issue tracker 的实际 label 字符串。

| mattpocock/skills 中的 label | 本仓库 tracker 中的 label | 含义 |
|---|---|---|
| `needs-triage` | `needs-triage` | 需要 maintainer 评估该 issue |
| `needs-info` | `needs-info` | 等待 reporter 补充信息 |
| `ready-for-agent` | `ready-for-agent` | 规格完整，可交由 AFK agent 处理 |
| `ready-for-human` | `ready-for-human` | 需要人工实现 |
| `wontfix` | `wontfix` | 不会处理 |

当 skill 提到某个 role（例如“apply the AFK-ready triage label”）时，使用表中对应的 label 字符串。

如需适配已有 vocabulary，编辑右侧 label 列。
